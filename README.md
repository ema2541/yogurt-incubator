# Yogurt Incubator

Arduino UNO R3 -> MAX6675
- 5V -> VCC
- GND -> GND
- 10 -> SCK
- 9 -> CS
- 8 -> SO
- 2 -> Output to activate fan.
- 4 -> Output to activate heater.
- fan and heater connect to ground and 5v
